* Goal
Render a grid of purchasable items when the Shop tab is active. Items display
thumbnails, titles, descriptions, prices, and sales counts. Owned items show
download buttons instead of prices.

* Reference
~.claude/references/shop-about-reference.html~ — Lines 737-894

* Data Model
** Schema: sort options
Ident entities for sort. No labels — those come from I18n.

#+begin_src clojure
{:db/ident       :shop.sort/newest
 :sort/attribute :post/created-at
 :sort/direction :sort.direction/desc}

{:db/ident       :shop.sort/popular
 :sort/attribute :post/sold-count
 :sort/direction :sort.direction/desc}

{:db/ident       :shop.sort/price-asc
 :sort/attribute :post/price
 :sort/direction :sort.direction/asc}
#+end_src

** Schema: media types
#+begin_src clojure

** Ownership
User has purchases (relationship from owner's perspective):

#+begin_src clojure
{:db/ident       :user/purchases
 :db/valueType   :db.type/ref
 :db/cardinality :db.cardinality/many}
#+end_src

Each purchase links to post and records timestamp:

#+begin_src clojure
{:purchase/post :post/id
 :purchase/at   #inst "..."}
#+end_src

Query: does user own post?
#+begin_src clojure
(d/q '[:find ?purchase .
       :in $ ?user ?post
       :where
       [?user :user/purchases ?purchase]
       [?purchase :purchase/post ?post]]
     db user-id post-id)
#+end_src

* Components

** shop-toolbar
Sort buttons use ~data-action~. Active sort has accent styling.

** shop-item
Purchasable items use ~bits.form/form~ for CSRF protection. Owned items show
badge and download button.

** shop-grid
Responsive 2-column grid, single column on mobile.
