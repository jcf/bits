* Goal
Pure Hiccup components rendering the creator profile with Tailwind semantic
classes. No inline hex codes, no positional arguments with naked numbers.

* Architecture
All components are pure functions taking a single data map.

#+begin_src clojure
;; Good: single map argument
(defn profile-header
  [{:keys [creator viewer-count stats]}]
  ...)

#+end_src

*** View
#+begin_src clojure
(defn creator-profile-view
  [request]
  (let [creator (:session/creator request)
        data    {:creator      creator
                 :viewer-count (count-viewers (:creator/handle creator))
                 :stats        {:price           "£10"
                                :supporter-count 23
                                :post-count      12}}]
    (list
     (bits-bar {:request request})
     (banner {:creator creator})
     (profile-header data)
     (feed-tabs {:active-tab :posts}))))
#+end_src
