* Goal
Pure Hiccup components rendering the creator profile with Tailwind semantic
classes. No inline hex codes, no positional arguments with naked numbers.

*** View
#+begin_src clojure
(defn creator-profile-view
  [request]
  (let [creator (:session/creator request)
        data    {:creator      creator
                 :viewer-count (count-viewers (:creator/handle creator))
                 :stats        {:price           "£10"
                                :supporter-count 23
                                :post-count      12}}]
    (list
     (bits-bar {:request request})
     (banner {:creator creator})
     (profile-header data)
     (feed-tabs {:active-tab :posts}))))
#+end_src

* Routing
Subdomain-based only. When ~:session/realm~ contains ~:creator/handle~ (from
~wrap-realm~ middleware), render creator profile at ~/~.

#+begin_src clojure
;; In bits.next, the "/" handler checks for creator data
(defn- creator-realm? [request]
  (some? (:creator/handle (:session/realm request))))
