* Justfile additions
Add to ~[group('docs')]~ section:

** just prompt
Create a new prompt document in ~.claude/prompts/~.

#+begin_src just
# Create a new prompt
[group('docs')]
prompt +title:
    #!/usr/bin/env zsh
    timestamp=$(date +%Y%m%d%H%M%S)
    normalized=$(echo "{{ title }}" | tr '[:upper:]' '[:lower:]' | sed 's/[^a-z0-9]\+/-/g' | sed 's/^-\|-$//g')
    filename=".claude/prompts/${timestamp}-${normalized}.org"
    mkdir -p .claude/prompts
    cat > "$filename" <<EOF
#+title:    {{ title }}
#+date:     $(date +%Y-%m-%d)
#+status:   todo
#+property: header-args :dir ../..
EOF
    echo "💭 {{ BOLD }}Created \"$filename\"{{ NORMAL }}."
#+end_src

** just execute
Interactive fzf-based prompt selector. Uses the script from the original message
with status indicators, preview, and filtering.

** .claude/skills/execute/SKILL.md
#+begin_src markdown
---
name: execute
description: Select and execute a prompt
allowed-tools: Read, Glob, Bash, Edit, Write
---

Execute a prompt from `.claude/prompts/`.

## Usage

- `/execute` — Interactive: list prompts, user selects via AskUserQuestion
- `/execute <slug>` — Direct: execute the specified prompt

## Process

1. If no argument: glob `.claude/prompts/*.org`, present options via AskUserQuestion
2. Read the selected prompt file
3. Parse status from frontmatter (skip if status: done, unless --all)
4. Execute according to the prompt's instructions
5. Update prompt status as work progresses
#+end_src
