Within Bits we have two modes of operation: colo and solo.

Colo (or co-location) maps host domains to corresponding hosts. For example:

1. alice.bits.page
2. bob.bits.page
3. charlie.shop
4. bits.milly.co.uk

Solo simplifies things by routing all incoming requests to the same tenant.
